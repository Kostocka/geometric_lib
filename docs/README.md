|:----------------------|:---------------:|--------------:|
| Circle -> area        |        3        |     28.274    |
| Rectangle -> perimeter|      10 4       |      28       |
| Square -> perimeter   |       100       |     400       |
| Triangle -> area      |      10 4       |      20       |

## Mathematical formulas for calculation
### Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²
- Triangle: S = ½ * a * h

### Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a + b + c
